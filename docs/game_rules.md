# 📜 Правила игры и Баланс (game_rules.md)

В данном документе описаны игровые механики гибридной системы «Шахматы против Шашек», система штрафов и условия завершения партии.

---

## 1. Основная концепция
Игра представляет собой асимметричное противостояние:
* **Белые (Шахматы):** Обладают высокой мобильностью и дальнобойностью, но крайне ограничены во времени и уязвимы к штрафам.
* **Черные (Шашки):** Имеют преимущество в количестве фигур и времени, но ограничены в перемещении (только темные клетки) и дальности атаки.

## 2. Временной ресурс
Время — это единственный «HP» игрока. Начальные лимиты:
* **Шахматы:** 31 секунда.
* **Шашки:** 240 секунд.

## 3. Система штрафов (Chess Penalties)
Потеря шахматных фигур приводит к мгновенному вычитанию секунд из таймера белых. Штрафы определены в `ChessPieceType`:

| Фигура | Тип в коде | Штраф (сек) | Описание |
| :--- | :--- | :--- | :--- |
| **Пешка** | `PAWN` | -1 с | Расходный материал. |
| **Конь** | `KNIGHT` | -10 с | Высокая цена за тактическую единицу. |
| **Слон** | `BISHOP` | -10 с | Потеря контроля над диагональю. |
| **Ладья** | `ROOK` | -10 с | Серьезный удар по обороне. |
| **Ферзь** | `QUEEN` | -20 с | Критическая потеря, часто ведущая к поражению. |
| **Король** | `KING` | 0 с | Потеря короля означает немедленный конец игры. |

> **Особый штраф:** Объявление **Шаха** белому королю списывает с белых **20 секунд**.

---

## 4. Условия победы и поражения

### **Победа Шахмат (Белых):**
1. **Тотальная зачистка:** На доске не осталось ни одной черной шашки.
2. **Блокировка (Пат):** У черных остались фигуры, но нет ни одного легального хода (все заблокированы).
3. **Время вышло:** Таймер черных достиг отметки 00:00.

### **Победа Шашек (Черных):**
1. **Убийство Короля:** Белый король захвачен черной шашкой.
2. **Время вышло:** Таймер белых достиг 0 (из-за раздумий или накопившихся штрафов).
3. **Критическое доминирование:** Черные смогли превратить **две шашки в дамки** одновременно. Согласно логике ИИ, эта ситуация считается безнадежной для шахмат.

---

## 5. Механики фигур

### **Шашки (Black):**
* Ходят и бьют только по темным клеткам.
* **Взятие обязательно.** Если есть возможность бить — шашка должна это сделать.
* **Дамка:** Появляется при достижении 7-го ряда. Может перемещаться на любое количество клеток по диагонали.

### **Шахматы (White):**
* Ходят согласно классическим правилам (Ладья — по прямым, Слон — по диагоналям и т.д.).
* **Безопасная зона:** Белые клетки являются недосягаемыми для обычных шашек.
* **Превращение:** Пешка, достигшая 0-го ряда, превращается в Ферзя.
* **Рокировка:** В данной версии не предусмотрена для упрощения логики ИИ.
